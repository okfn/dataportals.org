1. Add an Individual Portal

You can add a portal by:

  1. Submitting an issue on Github
  2. Using Data Explorer
  3. Submitting a pull request on GitHub

How to Submit a Portal as a Github Issue

Press the button below to get started. This will redirect your browser to a page that will create a new Github issue for review.

Add a Portal

What happens next?

The issue will be reviewed by and incorporated into DataPortals.org pending no data errors.

We can always use some help. Editors review submissions for accuracy. You can choose to focus on a country or region to make the task more manageable. Every little contribution helps us build a comprehensive list of open data portals around the world.
